## 📌 Overview

This directory contains the deliverables for the **AI Fluency Assignment 1: Frame It as Cases**. The objective of this assignment is to draft a professional **Voice Card**, a concise **Bio** targeting early-stage SaaS and AI founders, and three detailed **Case Studies** describing self-directed projects (Orchard Bliss, Lumina3D, and Memory Card). This documentation is designed to showcase engineering rationale, design decisions, and problem-solving abilities directly to prospective clients and recruiters.

---

## ⚙️ How It Works

The case study documentation process is structured through the following stages:

| Step | Stage | Description |
|------|-------|-------------|
| 1 | **Voice Card** | Defining a professional, clear, honest, and concise communication style. |
| 2 | **Bio Creation** | Developing a clear value proposition statement aimed at early-stage startup founders. |
| 3 | **Case Studies** | Structuring three real project write-ups detailing the goals, key decisions, results, and learnings. |
| 4 | **Refinements** | Framing the copy to emphasize outcomes and lessons learned rather than just listing technical features. |
