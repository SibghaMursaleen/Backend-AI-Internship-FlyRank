## 📌 Overview

This directory contains the deliverables for the **AI Fluency Assignment 2: The Prompt Ladder**. This assignment demonstrates a structured approach to generating landing pages (such as Loom, Dial, Anchor, Relay, Foyer, and Foundry) by systematically layering instructions in prompts. The prompt ladder guides the LLM from a generic layout to a highly specific, audience-focused, and self-critiqued startup landing page.

---

## ⚙️ How It Works

The prompt layering technique is structured in a ladder format:

| Step | Stage | Description |
|------|-------|-------------|
| 1 | **Baseline Prompt** | Asking the AI to build a generic landing page with minimal guidance. |
| 2 | **Layering Audience** | Directing the output specifically toward early-stage SaaS and AI founders. |
| 3 | **Layering Context** | Explaining the project's purpose as a portfolio showcase to build credibility. |
| 4 | **Layering Format** | Defining the exact layout structure including Hero, Features, Pricing, and FAQ. |
| 5 | **Layering Quality** | Enforcing strict writing rules to ensure high-quality copy and remove AI fluff. |
| 6 | **Self-Review** | Instructing the model to evaluate its own output and generate self-improvement lists. |
