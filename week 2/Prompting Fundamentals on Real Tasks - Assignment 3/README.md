## 📌 Overview

This directory contains the deliverables for the **AI Fluency: Prompting Fundamentals on Real Tasks** assignment. This assignment details the iterative application of primary prompting techniques to design a clean landing page for a car showroom website named **Al-Faisal Motors**. Additionally, it presents a cross-model comparative analysis between **Claude** and **ChatGPT** to highlight differences in visual hierarchy, design-focus, and reasoning depth.

---

## ⚙️ How It Works

The prompt construction and execution process followed these distinct phases:

| Step | Stage | Description |
|------|-------|-------------|
| 1 | **Naive Prompting** | Testing the base model response with a simple request to build a landing page. |
| 2 | **Role Assignment** | Assigning an expert frontend designer role to improve visual aesthetics. |
| 3 | **Context & Motivation** | Providing the portfolio target and user motivations to align testimonials. |
| 4 | **Few-shot Examples** | Incorporating specific CTA and hero copy examples to guide style. |
| 5 | **Output Structuring** | Organizing the content sections logically (Hero, Features, Pricing, FAQ, CTA). |
| 6 | **Step Decomposition** | Splitting the prompt into discrete steps: Audience, Sections, Copy, and Review. |
| 7 | **Model Comparison** | Reviewing and documenting the design-focused vs. explanatory outputs of Claude and ChatGPT. |
